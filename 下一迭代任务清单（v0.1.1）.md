## Day 2：桌面端自动拉起后端（Sidecar）

- [ ] 在 Tauri 启动阶段自动检测 `127.0.0.1:8008` 是否可用。
- [ ] 若不可用，则自动启动 Python sidecar（`uvicorn local_review_copilot.app:app`）。
- [ ] 优先使用 `backend/.venv/Scripts/python.exe`，否则回退 `python`。
- [ ] 应用退出时清理本次拉起的后端进程（避免孤儿进程）。
- [ ] 增加失败提示：后端未成功启动时给出明确排查信息。

**当日验收**
- 不再需要手动开后端终端；
- 重启应用不出现重复后端进程。

---

## Day 3：目录选择器 + 文件勾选注入

- [ ] 增加“选择目录”按钮（桌面原生文件夹选择）。
- [ ] 扫描后展示文件列表（路径/类型/大小），支持滚动查看。
- [ ] 支持单个勾选、全选、清空。
- [ ] Chat/Review/Quiz 请求统一携带 `selected_paths`。
- [ ] UI 显示“已选择 N 个文件”。

**当日验收**
- 可以不手动输入目录路径；
- 三模式都能基于“已勾选文件”运行。

## Day 4：模型配置面板（可视化）

- [ ] 后端新增：
  - `GET /config`：读取当前运行配置；
  - `POST /config`：更新并持久化配置（`config.yaml`）。
- [ ] 前端新增“模型设置”卡片：
  - provider、endpoint、model、timeout、max_context_tokens、api_key_env。
- [ ] 保存后立即生效（重建 LLM client）。
- [ ] 异常配置给出可读错误提示。

**当日验收**
- 不改文件也能切换模型参数；
- 设置保存后新请求按新参数执行。

- [ ] 启动桌面应用后可自动使用后端，无需手工开第二个终端。
